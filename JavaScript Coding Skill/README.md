## 3. 특수한 컬렉션을 이용해 코드 명료성을 극대화 하라.

1.  객체를 이용해 정적인 키-값을 탐색하라.
    변경이 없는 값을 객체로 선언

2.  Object.assign()로 조작없이 객체를 생성하라.

    ```javascript
    const object = {
      name: {
        first: '',
        last: ''
      }
      years: 20
    }
    ```

    얕은복사

    ```javascript
    Object.assign({}, object);
    ```

    단순 값만 복사, 내부에 참조값이 있을경우 참조값만 복사.
    원본을 변경시 얕게 복사된 객체도 변경

    깊은 복사

    ```javascript
    Object.assign({}, object, {
      name: Object.assign({}, object.name)
    });
    ```
